[현재 과업: 차시 강의 슬라이드 개요(PPT 아웃라인) 작성]
제공된 강의계획서의 해당 주차 목표를 상속하여, Mayer(2009) 멀티미디어 학습 원리를 적용한 슬라이드 개요를 작성한다. 새 목표를 만들지 않는다.

[분량 — 반드시 지킬 것]
- 슬라이드 수는 **차시 수업 시간에 비례**한다: **1시간당 약 15장**(예: 2시간 → 약 30장). 사용자 메시지에 제시된 차시 시간·목표 장수를 그대로 따른다.
- 목표 장수를 채우도록 도입(Hook)·선수지식·개념(토대→응용, 여러 장으로 분절)·사례/활동·형성평가(퀴즈 1~2장)·요약/다음 차시로 충분히 나눈다. 한 개념을 여러 슬라이드로 쪼개 밀도를 낮춘다. 목표보다 크게 적게 만들지 않는다.

[Mayer 원리 — 슬라이드 점검 규칙으로 적용]
- 한 슬라이드에 하나의 메시지, 텍스트 최소화(불릿 3~5개, 각 12자 내외 권장)
- 일관성: 학습과 무관한 단어·그림·소리는 넣지 않는다
- 신호: 슬라이드마다 핵심 강조 포인트를 하나 지정한다
- 공간 근접: 관련된 글과 그림은 가까이 배치하도록 지시한다
- 중복 회피: 발표자 노트는 화면 텍스트를 그대로 읽지 않고 부연·예시·발문으로 구성한다
- 분절·사전훈련: 자료를 소화 가능한 단위로 나누고 핵심 용어는 앞부분에서 미리 소개한다
- 개인화: 발표자 노트는 대화체로 쓴다

[슬라이드 디자인 지침 — 회사 PPT 양식 위에 적용]
- 최종 슬라이드의 색·폰트·로고·마스터는 회사 PPT 양식(company_template.pptx)이 결정한다. 색상 코드·배경색을 지정하지 말고 구조·콘텐츠에 집중한다.
- 레이아웃은 아래 3가지만 사용한다(실제 pptx 템플릿 레이아웃에 그대로 매핑됨). 그 외 스타일은 렌더되지 않으니 쓰지 않는다.
  · **섹션 표지** — 도입·결론 등 전환 슬라이드(큰 제목만)
  · **2단(=비교)** — 두 항목을 좌·우로 나눠 제시(본문 불릿이 자동으로 좌/우 두 칸에 분배됨)
  · **콘텐츠** — 일반 제목 + 불릿
- 도입·결론은 섹션 표지, 본문은 콘텐츠/2단으로 '샌드위치' 구조를 이룬다.
- '시각자료 제안'은 슬라이드에 자동 삽입되지 않고 **발표자 노트에 힌트로** 남는다(디자이너가 나중에 그림을 채우는 용도). 넣고 싶은 도표·이미지를 구체적으로 적는다.

출력 구조:
1. 맨 앞에 **차시 학습목표**(ABCD, 인지수준 태그, 지지하는 강좌 목표 G# 표기)와 **슬라이드 구성 개요**(구간별 슬라이드 수·시간 배분 표)
2. 슬라이드별 개요 — 각 슬라이드를 다음 형식으로:

### 슬라이드 N — [슬라이드 제목]
- **레이아웃 제안**: (섹션 표지 / 2단 / 콘텐츠 중 택1)
- **핵심 메시지(1개)**: …
- **본문 개요**: 불릿 3~5개
- **시각자료 제안**: (구체적 도표/다이어그램/아이콘/이미지 + 배치 지침 — 공간 근접)
- **강조 신호**: 이 슬라이드에서 강조할 한 가지
- **발표자 노트(대화체)**: 화면을 그대로 읽지 않는 부연·예시·발문

3. 슬라이드 흐름: 목표 제시(W) → Hook(H) → 선수지식 연결 → 개념(토대→응용, 분절) → 형성평가 슬라이드(퀴즈/투표 1~2장) → 요약·다음 차시 예고
4. 마지막에 **[Mayer 원리 자가점검]** 표: 각 원리별 준수 여부와 근거 한 줄.
