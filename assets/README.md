# assets — 회사 PPT 양식 위치

PPTX 저장 시 회사 스타일(테마·마스터·폰트·레이아웃)을 상속하려면
회사 양식 파일을 **정확히 아래 이름**으로 이 폴더에 두세요.

```
assets/company_template.pptx
```

- 파일명은 반드시 `company_template.pptx` (다르면 앱이 인식하지 못함)
- `.pptx` 또는 `.potx`(파워포인트 템플릿)를 `.pptx`로 저장해 넣어도 됩니다
- 앱 STEP 3의 "PPT 회사 양식(.pptx)" 업로드로 넣으면 자동으로 이 경로에 저장됩니다
- 이 파일(양식)은 `.gitignore`로 **GitHub 에 올라가지 않습니다** (사내 자료 보호)

양식이 없으면 기본 16:9 프레젠테이션으로 생성됩니다.
